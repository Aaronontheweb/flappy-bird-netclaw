# 🐦 Flappy Bird Clone

A single-file Flappy Bird game built entirely with **NetClaw** ([netclaw.dev](https://netclaw.dev)) using a **Qwen3.6-35B-A3B** local model.

## How It Was Made

This entire game — HTML, CSS, JavaScript, and even the gameplay video — was created through natural language prompts to NetClaw, an open-source agentic infrastructure for running AI coding agents locally.

### Stack
- **AI Agent**: NetClaw v0.14.3
- **Model**: Qwen3.6-35B-A3B (FP8)
- **Output**: Single `index.html` file — zero dependencies
- **Hosting**: GitHub Pages

## About NetClaw

[NetClaw](https://netclaw.dev) is self-hosted agentic infrastructure built with .NET that lets you run AI coding agents locally with full tool access — including browser automation, file editing, video processing, and code execution. It's designed for developers who want to leverage LLMs for real work while keeping everything private and under their control.
